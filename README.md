## Run Locally

Clone the project

```bash
  git@github.com:NikolaM-Dev/counter-tsx.git
```

Go to the project directory

```bash
  cd counter-tsx
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  # Development
  yarn start:dev

  # Build
  yarn build

  # Preview
  yarn preview
```
